Ayuda - Importación y Exportación de Datos

Este formulario te permite gestionar la importación y exportación de datos de varias entidades (clientes, empleados, productos, coches). A continuación, se describen las funciones de cada uno de los botones disponibles.

Refrescar Lista

Refrescar la lista de objetos te permite actualizar los datos mostrados en la interfaz.

  • Haz clic en el botón para recargar la lista de objetos.

Importar/Exportar Datos de Clientes

Clientes te permite cargar o grabar datos de clientes desde o hacia archivos JSON y XML.

  • Cargar desde JSON: Haz clic en el botón para cargar los datos de clientes desde un archivo JSON.
  • Grabar en JSON: Haz clic en el botón para grabar los datos de clientes en un archivo JSON.
  • Grabar en XML: Haz clic en el botón para grabar los datos de clientes en un archivo XML.
  • Cargar desde XML: Haz clic en el botón para cargar los datos de clientes desde un archivo XML.
