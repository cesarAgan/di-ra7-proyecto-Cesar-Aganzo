Ayuda - Actualización y Eliminación de Objetos

Este formulario te permite modificar, eliminar o consultar objetos existentes en el sistema, como clientes, coches, empleados y productos. A continuación, se describen los controles disponibles con sus respectivos tooltips.

Botones Principales

  • Refrescar Lista: Botón para refrescar la lista de objetos. [tooltip]
  • Aceptar: Botón para ver un objeto en detalle. [tooltip]
  • Cancelar: Botón para cancelar la modificación o eliminación de objetos. [tooltip]
  • Guardar: Botón para guardar la modificación o eliminación de un objeto. [tooltip]

Selección de Tipo de Objeto

  • Selector Cliente: Radio button para seleccionar el tipo de objeto cliente. [tooltip]
  • Selector Coche: Radio button para seleccionar el tipo de objeto coche. [tooltip]
  • Selector Empleado: Radio button para seleccionar el tipo de objeto empleado. [tooltip]
  • Selector Producto: Radio button para seleccionar el tipo de objeto producto. [tooltip]

Acciones de Modificación y Eliminación

  • Eliminar por 'ID': Radio button para eliminar un objeto por su identificador único. [tooltip]
  • Modificar por 'ID': Radio button para modificar un objeto por su identificador único. [tooltip]
  • Consultar por 'ID': Radio button para consultar un objeto por su identificador único. [tooltip]

Controles Adicionales

  • Campo Nombre o Marca: Campos de texto para introducir información adicional, siempre seleccionando previamente el tipo de objeto.
  • Campo Apellido o Correo o Modelo: Campos de texto para introducir información adicional, siempre seleccionando previamente el tipo de objeto.
  • Campo descripcion: Campo para ingresar una descripción detallada del objeto (por ejemplo, producto).
  • Selector (VIP / Descapotable / Jornada reducida): Marca esta casilla para activar características especiales, como VIP , descapotable o jornada reducida.
  • Campo para fecha: Permite seleccionar una fecha. El campo es relevante según el tipo de objeto seleccionado.
  • Campo numéricos (Sueldo / Precio / Descuento): Permite ingresar valores numéricos para sueldo (empleado), precio (coche, producto) o descuento (producto).
  • Selector de categoría: Selecciona la categoría del empleado (por ejemplo, gerente, técnico).
  • Tooltips: Cada campo tiene una descripción emergente (tooltip) para ayudarte a completar los campos correctamente.

Recuerda que antes de modificar, eliminar o consultar un objeto, debes seleccionar el tipo de objeto correspondiente (cliente, coche, empleado o producto) y proporcionar el identificador adecuado.
