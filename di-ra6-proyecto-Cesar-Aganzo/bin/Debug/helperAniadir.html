Ayuda - Formulario de Añadir Objetos

Este formulario te permite añadir nuevos objetos al sistema, incluyendo clientes, coches, empleados y productos. A continuación, encontrarás una descripción de las funciones de cada uno de los botones y controles disponibles.

Botones Principales

  • Refrescar Lista: Actualiza la lista de objetos mostrados.
  • Aceptar: Añade un nuevo objeto (cliente, coche, empleado o producto) al sistema.
  • Limpiar Campos: Borra todos los campos para comenzar de nuevo.
  • Cancelar: Cancela la creación de un objeto nuevo y cierra el formulario.
  • Guardar: Guarda la creación del nuevo objeto, registrando toda la información en el sistema.

Selección de Tipo de Objeto

  • Selector Cliente: Selecciona este radio button para crear un cliente.
  • Selector Coche: Selecciona este radio button para crear un coche.
  • Selector Empleado: Selecciona este radio button para crear un empleado.
  • Selector Producto: Selecciona este radio button para crear un producto.

Controles Adicionales

  • Campo Nombre o Marca: Campos de texto para introducir información adicional, siempre seleccionando previamente el tipo de objeto.
  • Campo Apellido o Correo o Modelo: Campos de texto para introducir información adicional, siempre seleccionando previamente el tipo de objeto.
  • Campo descripcion: Campo para ingresar una descripción detallada del objeto (por ejemplo, producto).
  • Selector (VIP / Descapotable / Jornada reducida): Marca esta casilla para activar características especiales, como VIP , descapotable o jornada reducida.
  • Campo para fecha: Permite seleccionar una fecha. El campo es relevante según el tipo de objeto seleccionado.
  • Campo numéricos (Sueldo / Precio / Descuento): Permite ingresar valores numéricos para sueldo (empleado), precio (coche, producto) o descuento (producto).
  • Selector de categoría: Selecciona la categoría del empleado (por ejemplo, gerente, técnico).
  • Tooltips: Cada campo tiene una descripción emergente (tooltip) para ayudarte a completar los campos correctamente.

Recuerda que debes seleccionar primero un tipo de objeto (cliente, coche, empleado o producto) antes de completar la mayoría de los campos en el formulario.
